# OpenGL 3.2+ extensions:
GL_EXT_texture_compression_s3tc - hard requirement
GL_ARB_texture_filter_anisotropic or GL_EXT_texture_filter_anisotropic - hard requirement
GL_ARB_compute_shader - soft requirement
GL_ARB_shader_image_load_store or GL_EXT_shader_image_load_store - soft requirement
GL_ARB_shading_language_420pack - soft requirement
GL_ARB_arrays_of_arrays - soft requirement
GL_ARB_enhanced_layouts - soft requirement


* hard - we can NOT run the application without that extension
* soft - we can run the appplication without that extension
